# EndProject
End Project - Frontend
Lyrics Finder

Lyrics Finder är en enkel webbaserad applikation som låter användare söka efter låttexter. Genom att skriva in artist och låtnamn hämtas texten direkt från ett API och visas på sidan.

Funktioner

Sök efter låttext genom att ange artist och låt.

Felhantering om låttext inte finns eller om något går fel vid API-anropet.

Användarvänlig visning av låttext i en låttext-box.

Responsiv design.

Tekniker

HTML, CSS och JavaScript

API: lyrics.ovh
för att hämta låttexter
